# 安裝 Markdown 渲染優化依賴

## 需要安裝的包

```bash
npm install react-syntax-highlighter @types/react-syntax-highlighter remark-gfm
```

或使用 yarn:

```bash
yarn add react-syntax-highlighter @types/react-syntax-highlighter remark-gfm
```

## 包說明

- **react-syntax-highlighter**: 代碼語法高亮（ChatGPT/Gemini 都使用）
- **@types/react-syntax-highlighter**: TypeScript 類型定義
- **remark-gfm**: GitHub Flavored Markdown 支持（表格、任務列表等）

## 安裝後重啟開發服務器

```bash
npm run dev
```
